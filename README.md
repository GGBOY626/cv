## 🛠 Installation & Set Up

1. Install the Gatsby CLI

   ```sh
   npm install -g gatsby-cli
   ```

2. Install dependencies

   ```sh
   yarn
   ```

3. Start the development server

   ```sh
   npm start
   ```

## 🚀 Building for Production

```sh
npm run build
npm run serve
```
